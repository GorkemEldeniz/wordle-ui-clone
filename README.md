# React + TypeScript + Vite + TailwindCSS

### The worlde ui clone with react

- Clone the repo

```bash
   git clone https://github.com/GorkemEldeniz/wordle-ui-clone
```

- Open the main folder

```bash
   cd wordle-ui-clone
```

- Install dependencies

```bash
   pnpm install
```

- Run the project on your local

```bash
   pnpm run dev
```
